VLC Settings

Configure how VLC media player behaves when launching movies.

When enabled, all existing VLC windows will be closed before launching a new movie. When disabled, new movies will open in addition to any existing VLC windows.
When enabled, automatically enable subtitles if a subtitle file is found in the same folder as the movie. When disabled, movies will launch without subtitles unless manually selected.
Use GPU hardware decoding for faster video rendering. May cause issues on some systems. Only affects movies launched from Movie Searcher.

What does this do?

Command-line optimizations (always active):

  • Reduced file caching (1200ms ‚Üí 300ms)
  • Fast seeking for timestamps
  • Disabled metadata network lookups
  • Disabled auto-preparsing

System-wide optimization (optional):

Modifies VLC's config file (vlcrc) to apply optimizations to ALL VLC usage. Creates a backup first. These settings affect VLC even when launched outside of Movie Searcher.

Note: If VLC startup is still slow, check if your movie drive is an HDD that may need spin-up time, or if antivirus is scanning files.
